インプレスジャパン発行「上を目指すプログラマーのためのiPhoneアプリ開発テクニック iOS 7編」サンプルコード【概要】Text Kitを利用したマルチカラム表示の電子書籍風サンプルプロジェクトです。Xcodeの"Page-Based Application"のテンプレートを元に作成します。テンプレート作成後に変更したファイルは以下になります。・ModelController.mNSTextStorage, NSLayoutManager, NSTextContainerを利用して表示内容を管理・DataViewController.h/m左右のUITextViewをプロパティとして追加し、viewDidLoadで内容を表示【補足事項】本サンプルプロジェクトは、NSTextContainerとNSLayoutManagerを連携させてマルチカラムレイアウトを実現する例を示したものです。デバイスの回転処理等、細かい処理には対応していません。DataViewController内で読み込んでいるテキストファイル(sample-text.txt)は、青空文庫よりダウンロードしたものです。【実行環境】iOS SDK : 7.0.3Xcode : 5.0.2iOS : 7.0以降
